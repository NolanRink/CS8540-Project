# Phase 1 

---

## Goal

1. Use the tweets dataset(s) provided on Canvas.
2. Write your own code (any language) to extract **all hashtags and URLs** from the tweets.
3. Run the standard WordCount example **twice**:
   * once with **Apache Hadoop**,
   * once with **Apache Spark**
   on the extracted hashtags/URLs data.
4. Collect the output and log files generated by Hadoop and Spark.
5. Execute the above steps on **FABRIC**.
6. Add a README file (that’s this document).


## Hadoop commands executed

The following sequence of shell commands was used to run the Hadoop portion of the project:

```sh
git clone https://github.com/NolanRink/CS8540-Project.git

cd CS8540-Project/
unzip tweets.json.zip

python3 phase1/extract_hashtags_urls.py

hdfs dfs -mkdir project
hdfs dfs -put phase1/extracted/combined.txt /project/combined.txt

hadoop jar $HADOOP_HOME/share/hadoop/mapreduce/hadoop-mapreduce-examples-*.jar \
    wordcount /project/combined.txt /project/out_wc_hadoop

hdfs dfs -get /project/out_wc_hadoop /home/ubuntu/CS8540-Project/phase1
```
